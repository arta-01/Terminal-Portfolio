## Running Locally

Clone the project

```bash
git clone https://github.com/arta-01/Terminal-Prtfolio
```

Go to the project directory

```bash
cd terminal-portfolio
```

Remove remote origin

```bash
git remote remove origin
```

Install dependencies

```bash
npm install
```

Start the server

```bash
npm run dev
```
